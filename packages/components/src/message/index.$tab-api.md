## API

| 属性名 | 描述 | 类型 | 默认值 |
| --- | --- | --- | --- |
| className | 自定义 CSS class | `string` | - |
| content | 提示内容 | `ReactNode` | - |
| duration | 自动关闭的延时，单位豪秒。| `number` | - |
| key | 当前提示的唯一标志 | `number` |-|
| style | 自定义内联样式 | `CSSProperties` |-|
| onClose | 关闭时触发的回调函数 | `function` |-|
